# GuardDuty

## Status

GuardDuty was **not** enabled in this project due to account limitations.

## Impact

Although GuardDuty was not available, the environment still included core visibility through:

- CloudTrail
- AWS Config
- CloudWatch

This ensured that security-relevant configuration changes and management activity could still be documented and investigated.

## Note

This project acknowledges the limitation and treats it as part of the documented scope of the environment rather than attempting to imply coverage that was not present.
